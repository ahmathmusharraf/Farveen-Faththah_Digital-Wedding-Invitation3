## 📖 Project Overview

Farveen & Faththah Wedding Digital Invitation System is a modern, elegant, and fully responsive wedding invitation website designed to provide guests with a seamless digital invitation experience.

The platform replaces traditional printed invitations with an interactive web-based solution that allows guests to view event details, explore the couple's story, access venue information, and confirm attendance digitally.

Inspired by modern digital invitation platforms that emphasize RSVP management, personalized guest experiences, mobile-first design, and event information accessibility, the system focuses on simplicity, elegance, and user engagement.
